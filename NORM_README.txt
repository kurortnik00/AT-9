1) Установить MasterScada.
2) Установить OPCserver.
3) Отрыть файл "OPCserver\data.mbp", запустить его (нажать на кнопку плей в левом верхнем углу, в отчёте в самом конце есть скрины). Это делается для того чтоб создались файлы в папке ProgramData.
4) Скопировать файл "ini.txt" в папку "C:\ProgramData\InSAT\MasterOPC Universal Modbus Server\SERVERCFG".
5) Теперь запустив сервер должно быть инициализированно 120 переменных значениями из txt файла.
6) Открыть проект в MasterScada "AT-9\AT-9.vav"
7) Нажать кнопку "ок" при запросе пароля имени пользователя, следующее окно сообщения просто закрыть.
8) Теперь можно полазить по блокам поглядеть что как работет и с чем связанно.
	- Можно открыть нагляжную мнемосхему, для этого щёлкнуть правой кнопкой мыши по "АТ-9" в контекстном меню выбрать "перейти на" -> "мнемосхему".
	- На мнемосхеме можно нажимать на макроблоки. 
10) Отчёт местами хорош, местами сделан в угоду зачёту (так как хочет препод). Ответсвенности за UML диаграммы не несу.
